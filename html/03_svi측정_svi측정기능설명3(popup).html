SVI 측정 네이게이션 영역으로 측정 전체정보와 현재위치 그리고 입력여부를 표시하며,
측정지표 항목 마우스 오버시 지표명이 표시되고 클릭시 해당 지표로 바로 이동합니다.

측정 항목(아이콘)에 대한 설명은 다음과 같습니다.

  • : SVI 측정 전체에서 현재 SVI 측정지표 위치를 의미합니다.

  • : 지표의 측정 항목을 모두 입력하고 작성내용검증이 완료됨을 의미합니다.

  • : 지표의 측정 항목을 하나이상 입력했지만 완료되지 않음을 의미합니다.

  • : 지표의 측정 항목을 아무것도 입력하지 않은 상태를 의미합니다.
